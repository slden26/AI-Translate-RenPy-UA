# AI Translate - Ren'Py UA
<p align="center"><img src="https://github.com/user-attachments/assets/df92a4b2-9753-4e2a-b580-91909a4e8a37" width="" alt=""></p>

Інструмент для автоматичного редагування та перекладу ігрових файлів Ren'Py (.rpy) за допомогою штучного інтелекту (через OpenRouter або інші API).

### Основні функції:
- **Пакетна обробка (Batch Size):** Налаштування кількості рядків для одного запиту до ШІ.
- **Контроль температури (Temperature):** Регулювання точності або творчості відповідей моделі.
- **Автоматичні налаштування:** Всі дані (API ключ, URL, моделі, шляхи) зберігаються у `settings.ini`.
- **Зручний інтерфейс:** Лаконічне вікно з фіксованим розміром.

### Реліз v1.1 — Denis Edition
#### Що нового:

- **Рекурсивний пошук:** Тепер програма автоматично знаходить усі .rpy файли не тільки в корінній папці, а й у всіх підпапках (зручно для великих проектів з розгалуженою структурою tl).
- **Захист від зміщення рядків:** Додано суворий парсинг за індексами (index:translation). Тепер, навіть якщо ШІ спробує додати від себе вступні фрази або коментарі, програма коректно розставить переклад по своїх місцях.
- **Покращений алгоритм очищення:** Автоматичне видалення сміття з відповідей ШІ (Markdown-розмітка, зайві лапки, префікси).
- **Оптимізований системний промт:** Новий базовий промт змушує модель перекладати кожен рядок без винятків, зберігаючи всі теги Ren'Py.
"You are a professional Ren'Py translator. Translate EVERY line provided by the user into the target language. Preserve tags like [] and {}. Maintain the format 'index:translation'. Do not skip any lines. Do not add comments or introduction."
- **Збереження структури:** Програма автоматично створює необхідні підпапки в директорії виводу, повторюючи структуру вхідної папки.
